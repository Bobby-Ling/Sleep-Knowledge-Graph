## 就诊流程
1. 一般资料
    年龄 身高 病史 ... -> 推理
2. 问卷/量表
    自评(稍后提供)
    关键词 评估
3. 诊断(根据1, 2)
    ... + 症状 -> 病症
    -> 并发症
    -> 建议(应做哪些检查/治疗{药物/非药物}/药物推荐(根据资料))
    -> 临床对比(真实病历)

## 目的: 初筛
- 缩短医生问诊过程(医院内)
- 社区医院/养老机构(复杂/重症/... -> 专业睡眠医学中心)

## 交互(重点)

界面与交互(字体/文本描述)
